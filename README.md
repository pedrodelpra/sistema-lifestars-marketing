# LifeStars — Marketing OS v2.0

Um sistema de gerenciamento de marketing premium, moderno e dinâmico desenvolvido para a **LifeStars Cuidadores**.

## 🚀 Como Executar

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Navegue até a pasta do projeto:
   ```bash
   cd sistema-marketing-lifestars
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
5. Abra o link exibido no terminal (geralmente `http://localhost:5173`).

## ✨ Funcionalidades

- **Dashboard Inteligente**: Visão geral de métricas, próximos conteúdos e checklist semanal.
- **Brand Bible**: Repositório central da identidade visual, missão e valores da marca.
- **Público & Personas**: Detalhamento dos perfis atendidos (Ana Paula, José Carlos, etc).
- **Calendário Editorial**: Gestão de posts para Instagram, Blog e Ads.
- **Banco de Clientes**: Controle de autorizações de imagem e histórico de publicações.
- **Design System Premium**: Interface baseada em Glassmorphism, tipografia moderna (Outfit) e animações fluidas.

## 🛠️ Tecnologias

- **Vite**: Build tool ultra-rápida.
- **Vanilla JS (ES6+)**: Lógica limpa e modular.
- **CSS3 Moderno**: Variáveis, Grid, Flexbox e efeitos de desfoque.
- **HTML5 Semântico**: SEO otimizado.
